# dropdown

Overlay dropdown utility for OneNote Ribbon.

## Exports

- `showDropdown(anchor, items, opts?)` — renders positioned dropdown overlay
- `DropdownItem` — interface for dropdown items
- `DropdownOpts` — optional styling overrides

## Tests

- `tests/dropdown.unit.ts` — basic render, item count, outside-click
- `tests/dropdown.edge.ts` — scroll overflow, viewport clamp, disabled items
