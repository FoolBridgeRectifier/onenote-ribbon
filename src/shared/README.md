# shared

Shared pure-logic utilities used across Home and Insert tab button modules.

| File                   | Exports                                        | Used by                                           |
| ---------------------- | ---------------------------------------------- | ------------------------------------------------- |
| `dropdown/Dropdown.ts` | `showDropdown`, `DropdownItem`, `DropdownOpts` | All dropdown buttons                              |
| `toggleInline.ts`      | `toggleInline`                                 | Bold, Italic, Underline, Strikethrough, Highlight |
| `toggleSubSup.ts`      | `toggleSubSup`                                 | Subscript, Superscript                            |
| `toggleLinePrefix.ts`  | `toggleLinePrefix`                             | BulletList, NumberedList                          |

## Tests

Run each test file via `mcp__obsidian-devtools__evaluate_script`.
