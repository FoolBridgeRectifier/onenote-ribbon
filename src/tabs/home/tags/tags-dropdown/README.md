# tags-dropdown/

Overlay dropdown showing all 29 tags with icon, label, and active-check indicator. Uses the shared `showDropdown` utility for the Customize Tags footer action.

## Files

| File                | Purpose                                        |
| ------------------- | ---------------------------------------------- |
| `TagsDropdown.ts`   | `TagsDropdown.show(anchor, app)` static method |
| `tags-dropdown.css` | Scoped dark-theme styles for the overlay       |
