# clipboard/tests/

Integration tests for the full Clipboard group rendered in the Home panel.

## Tests

| File                       | Description                                                               |
| -------------------------- | ------------------------------------------------------------------------- |
| `clipboard.integration.ts` | Verifies all 5 commands exist in DOM, group name, and Paste layout height |

## How to run

Paste the exported string into `mcp__obsidian-devtools__evaluate_script` after reloading the plugin.
