### Tag Types

| Type         | Tags                                                    | Key Rule                                                                    |
|--------------|---------------------------------------------------------|-----------------------------------------------------------------------------|
| MD Closing   | `**` `*` `~~` `==` `` ` ``                              | Prefer in plain context; upgrade to HTML equivalent when inside HTML context |
| HTML Closing | `<b>` `<i>` `<s>` `<u>` `<sub>` `<sup>`                | Always HTML; compounds freely with other HTML tags                           |
| HTML Span    | color, font-size, font-family, background, align        | Compoundable; same CSS property name ŌåÆ replace value, never double-wrap     |
| Single       | `- ` list, `# ` heading, `> ` quote, `margin-left` indent | Line-level; applied to whole line regardless of inline selection position |
| Special      | `- [ ]` checkbox, `> [!type]` callout, `#todo`, meeting-details | Block/page-level; unique rules per tag                              |

### toggleTag ŌĆö Add vs Remove Decision

`[text]` = current selection. Golden rule: do what Microsoft Word does.

| Selection State                       | Inline (MD / HTML / Span)               | Single / Special              |
|---------------------------------------|-----------------------------------------|-------------------------------|
| Cursor (zero-width)                   | Remove enclosing tag at cursor; if none, mark insertion format | Remove line tag from current line |
| Inert zone (fenced code / tab / math) | No-op                                   | No-op                         |
| Tag fully covers selection            | **Remove**                              | **Remove** from line          |
| Selection fully inside tag (partial)  | **Remove** ŌĆö punch-out selected portion | **Remove** from line          |
| Selection spans tag boundary          | **Add** ŌĆö extend tag to cover selection | N/A (line-level handles it)   |
| No tag in selection                   | **Add**                                 | **Add**                       |
| Multi-line: all lines tagged          | **Remove** from all                     | **Remove** from all           |
| Multi-line: some lines tagged (mixed) | **Add** to untagged lines (Word: make uniform) | **Add** to untagged lines |
| Multi-line: no lines tagged           | **Add** to all                          | **Add** to all                |

### On Add

| #   | Scenario                                         | Behavior                                                               | Before ŌåÆ After                                                                                       |
|-----|--------------------------------------------------|------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| A1  | MD tag, plain context                            | Wrap with MD delimiters                                                | `[hello]` ŌåÆ `**[hello]**`                                                                            |
| A2  | MD tag, HTML context (enclosing HTML present)    | Upgrade to HTML equivalent                                             | `<b>x</b> [hello]` ŌåÆ `<b>x</b> <b>[hello]</b>`                                                      |
| A3  | HTML closing tag                                 | Wrap with HTML tag                                                     | `[hello]` ŌåÆ `<b>[hello]</b>`                                                                         |
| A4  | Span tag, no enclosing same-property             | Wrap with new span                                                     | `[hello]` ŌåÆ `<span style="color:red">[hello]</span>`                                                |
| A5  | Span tag, same CSS property already enclosing    | Replace attribute value only ŌĆö no double-wrap                          | `<span style="color:red">[hello]</span>` ŌåÆ `<span style="color:blue">[hello]</span>`                |
| A6  | Span tag, different CSS property already enclosing | Compound: nest new span inside existing                              | `<span style="color:red">[hi]</span>` + font-size ŌåÆ `<span style="color:red"><span style="font-size:14pt">[hi]</span></span>` |
| A7  | Selection spanning tag boundary (partial overlap) | Extend existing tag to cover full selection                           | `**hel**[lo world]` ŌåÆ `**hel[lo world]**`                                                           |
| A8  | MD tag already present, HTML equivalent also present | Treat as already tagged ŌåÆ **Remove** path instead                 | (see R9)                                                                                             |
| A9  | Adjacent same-type tags crossing selection        | If all content covered ŌåÆ **Remove** path; if partial ŌåÆ extend/merge  | `**a** [b] **c**` all bold ŌåÆ Remove; `**a**[ b **c**]` partial ŌåÆ extend to `**a b c**`             |
| A10 | Protected range inside selection                  | Skip protected token; wrap non-protected segments separately           | `[[link]] [hello]` ŌåÆ `[[link]] **hello**`                                                           |
| A11 | Multi-line: none tagged                           | Add to all lines                                                       | `l1\nl2` (both selected) ŌåÆ `**l1**\n**l2**`                                                         |
| A12 | Multi-line: some tagged (mixed)                   | Add to untagged lines only                                             | `**l1**\nl2` ŌåÆ `**l1**\n**l2**`                                                                     |
| A13 | Single: list                                      | Prepend `- ` to each selected line                                     | `item` ŌåÆ `- item`                                                                                    |
| A14 | Single: heading (h1ŌĆōh6)                           | Prepend `# ` at specified level to each line                           | `title` ŌåÆ `# title`                                                                                  |
| A15 | Single: quote                                     | Prepend `> ` to each line                                              | `note` ŌåÆ `> note`                                                                                    |
| A16 | Single: indent                                    | Insert `<span style="margin-left:Npx"/>` after all other line prefixes; N += 24 per toggle | `- item` ŌåÆ `- <span style="margin-left:24px"/>item`               |

### On Remove

| #   | Scenario                                           | Behavior                                                              | Before ŌåÆ After                                                                                |
|-----|----------------------------------------------------|-----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| R1  | MD tag, selection matches tag exactly              | Remove both delimiters                                                | `**[hello]**` ŌåÆ `[hello]`                                                                     |
| R2  | MD tag, partial selection inside tag (punch-out)   | Split: un-tag selected portion; keep surrounding text tagged          | `**hel[lo]**` ŌåÆ `**hel**[lo]`                                                                 |
| R3  | MD tag, selection includes delimiters (delimiter-inclusive) | Remove delimiters cleanly                                    | `[**hello**]` ŌåÆ `[hello]`                                                                     |
| R4  | MD tag in HTML context; HTML equivalent present    | Remove HTML equivalent                                                | `<b>[hello]</b>` ŌåÆ `[hello]`                                                                  |
| R5  | HTML closing, full selection                       | Unwrap HTML tag                                                       | `<b>[hello]</b>` ŌåÆ `[hello]`                                                                  |
| R6  | HTML closing, partial selection (punch-out)        | Split: leave unselected portion still tagged                          | `<b>hel[lo]</b>` ŌåÆ `<b>hel</b>[lo]`                                                          |
| R7  | HTML span, property name match (any value)         | Remove span entirely                                                  | `<span style="color:red">[hi]</span>` ŌåÆ `[hi]`                                               |
| R8  | HTML span, partial selection inside span (punch-out) | Split span: un-span selected portion                                | `<span style="color:red">hel[lo]</span>` ŌåÆ `<span style="color:red">hel</span>[lo]`          |
| R9  | MD + HTML equivalent both enclosing               | Remove both                                                           | `<b>**[hello]**</b>` ŌåÆ `[hello]`                                                              |
| R10 | sub present, toggle sup                           | Swap entire tag `<sub>` ŌåÆ `<sup>` (mutual exclusion)                 | `<sub>[x]</sub>` + toggle sup ŌåÆ `<sup>[x]</sup>`                                             |
| R11 | sup present, toggle sub                           | Swap entire tag `<sup>` ŌåÆ `<sub>`                                     | `<sup>[x]</sup>` + toggle sub ŌåÆ `<sub>[x]</sub>`                                             |
| R12 | sub/sup partial selection                         | Punch-out selected portion from the swapped tag                       | `<sub>hel[lo]</sub>` + toggle sup ŌåÆ `<sup>hel</sup>[lo]`                                     |
| R13 | Stacked same-type (`<b><b>text</b></b>`)          | Remove all duplicate layers in one pass                               | `<b><b>[hello]</b></b>` ŌåÆ `[hello]`                                                           |
| R14 | Multi-line: all tagged                            | Remove from all                                                       | `**l1**\n**l2**` ŌåÆ `l1\nl2`                                                                  |
| R15 | Multi-line: mixed (some tagged)                   | Remove from tagged lines; leave untagged lines unchanged              | `**l1**\nl2` ŌåÆ `l1\nl2`                                                                       |
| R16 | Single: list                                      | Remove `- ` prefix                                                    | `- [item]` ŌåÆ `[item]`                                                                         |
| R17 | Single: heading                                   | Remove `# ` prefix                                                    | `# [title]` ŌåÆ `[title]`                                                                       |
| R18 | Single: quote                                     | Remove `> ` prefix                                                    | `> [note]` ŌåÆ `[note]`                                                                         |
| R19 | Single: indent                                    | Decrease margin-left by 24px; remove span entirely when reaching 0   | `<span style="margin-left:48px"/>[item]` ŌåÆ `<span style="margin-left:24px"/>[item]`          |

### Protected Ranges ŌĆö always skip

On **add**: punch out ŌĆö split the selection at the protected token boundary; apply the tag to non-protected segments on either side; the token itself is left untouched.
On **remove of the protected token itself**: reconcile the punch-out ŌĆö remove the token delimiters and merge adjacent same-type tags on both sides into a single tag.
On **remove of an outer tag that was punched out**: detect the punch-out pattern and remove that tag from all non-protected segments together.

| Token                     | On Add (punch out)                                                                 | On Remove token (reconcile)                                                | On Remove outer tag                                          |
|---------------------------|------------------------------------------------------------------------------------|----------------------------------------------------------------------------|--------------------------------------------------------------|
| Wikilink `[[...]]`        | Split at token: `**before**[[link]]**after**`                                      | Remove `[[` `]]` delimiters; merge adjacent same-type tags: `**beforelinkafter**` | Remove tag from both segments: `before[[link]]after`  |
| Embed `![[...]]`          | Same as wikilink                                                                   | Same                                                                       | Same                                                         |
| MD link `[t](url)`        | Same as wikilink                                                                   | Same                                                                       | Same                                                         |
| Code `` `...` ``          | Split at code span: `**before**\`code\`**after**`                                  | Remove backtick delimiters; merge adjacent same-type tags: `**beforecodeafter**` | Remove tag from both segments: `before\`code\`after`   |
| Todo `#todo`              | Split at token: `**before**#todo**after**`                                         | Remove `#todo` token; merge adjacent same-type tags: `**beforeafter**`    | Remove tag from both segments: `before#todoafter`            |
| Meeting details block     | Entire block is inert ŌĆö no inline or single tags applied inside or around it       | N/A                                                                        | No-op                                                        |

### removeAllTags (separate function)

Find all tags wrapping around (enclosing) the selection from outside + all tags fully inside the selection. Unwrap all from last-to-first (avoids offset drift). Deduplicate overlapping replacements. Single-line tags removed if line-start is within selection.

---

### copyFormat (separate function)

Capture all active and enclosing tags at cursor/selection ŌĆö not just enclosing, but also line-level tags active on the cursor line. Map each to its TagDefinition (span style ŌåÆ dynamic span definition). Detect domain (MD or HTML). Return `{ tagDefinitions[], domain }`. Include single-line tags if line-start is within selection.

On paste: reconcile the captured format with the new selection ŌĆö toggle any tags that differ between source and destination. Line-level tags (list, heading, indent) are applied at the start of each line in the new selection.

### Line Tag Precedence (on same line)

Highest to lowest. Applying a higher-precedence tag swaps out a lower-precedence one where they are mutually exclusive.

| Priority | Tag                 | Mutual Exclusion                              | Notes                                              |
|----------|---------------------|-----------------------------------------------|----------------------------------------------------|
| 1        | Callout `> [!type]` | Replaces or nests over any other line prefix  | Applying over another callout creates nested callout (X15) |
| 2        | List `- `           | Excludes heading and checkbox at prefix level | Can coexist inside callout (`> - item`)            |
| 3        | Indent              | ŌĆö                                             | In list context uses `\t`; outside list uses `margin-left` span |
| 4        | Heading `# `        | Excludes list and checkbox                    | Swaps with list (X4/X5)                            |
| 4        | Checkbox `- [ ]`    | Excludes list and heading                     | Same priority level as heading; replaces callout header (X16) |

### Invariants (never violate)

| # | Rule |
|---|------|
| I1 | **No stacked active tags** ŌĆö if a tag is already active on the selection, never wrap it again; remove or replace only |
| I2 | **No MD delimiters or line tags inside HTML context** ŌĆö `**` `*` `~~` `==` must never appear inside an enclosing HTML tag; always upgrade to the HTML equivalent (`<b>` `<i>` `<s>` `<span style="background:...">`) |
| I3 | **No inline tags inside code spans** ŌĆö code content is always inert; nothing wraps inside `` `...` `` |
| I4 | **Line tags at line start** ŌĆö single-line tags always operate on the line prefix position, never on mid-line inline content |
| I5 | **Indent in list = tab** ŌĆö inside a list item, indent is a `\t` character; outside a list, indent is `<span style="margin-left:24px"/>` |
| I6 | **Indent removed with list** ŌĆö when a list marker is removed from a line, all leading `\t` indent characters on that line are also removed |
